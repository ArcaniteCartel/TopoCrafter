## Building Legend Entries

### Feature Overview

Added a `showBuildings` toggle to the legend parameters. When enabled, the legend shows one entry per unique `(templateId, color)` combination found on the map. Entries whose user-facing labels are identical are merged into a single legend row showing multiple shape icons side by side with one description label.

Labels default to the full catalog template name; each can be overridden per `(templateId, color)` key in the parameter panel.

### Legend Config Changes

Added to `LegendConfig` in `types/index.ts`:

```typescript
showBuildings: boolean          // default: true
buildingLabels: Record<string, string>  // key: `${templateId}::${color}`
```

Added `templateId: string` to `BuildingEntry` so legend grouping has a stable key per placed building.

### Legend Overlay (LegendOverlay in MapCanvas.tsx)

`LegendOverlay` receives a new `buildings: BuildingEntry[]` prop. After the static items array is built, a dynamic block iterates placed buildings and:

1. Collects unique `(templateId, color)` pairs into a `seen` map
2. Groups entries that share the same label string into a `byLabel` map
3. Pushes one `{ type: 'building', label, buildingShapes: [...] }` item per label group

`buildingShapes` carries `{ shape, color, widthM, depthM }` for each entry in the group.

### Relative Scale Between Building Icons

All building shape icons in the legend share a single `buildingGlobalScale` derived from the real-world dimensions of all placed buildings. The largest building fills its slot; all others are drawn proportionally smaller.

Implementation: the `items.map` render loop is wrapped in an IIFE that computes `buildingGlobalScale` before the loop begins:

```typescript
{(() => {
  const bBh = rowH * 0.72
  let buildingGlobalScale = 1
  const allBuildingShapes = items.flatMap(it => it.buildingShapes ?? [])
  if (allBuildingShapes.length > 0) {
    let gs = Infinity
    for (const { widthM, depthM } of allBuildingShapes) {
      gs = Math.min(gs, sampW * 0.88 / widthM, bBh / depthM)
    }
    buildingGlobalScale = isFinite(gs) ? gs : 1
  }
  return items.map(({ type, label, color, buildingShapes }, i) => {
    // ...
    // inside building sample render:
    const pw = widthM * buildingGlobalScale
    const pd = depthM * buildingGlobalScale
    const d = buildingPath(shape, cx, midY, pw, pd)
  })
})()}
```
