## Project State at Session Start

All annotation tools share a unified drag/select system in `MapCanvas.tsx`. The tool panel (crosshair icon left of TopoCrafter logo) currently contains two tools:

| Tool | Icon | Enabled when |
|---|---|---|
| Elevation Flag | Pennant flag | Elevation calibration set (`calReady`) |
| Slope Arrow | Horizontal arrow | Calibration + map width set (`hasGroundResolution`) |

Key architectural patterns established:
- Annotations stored in Zustand (`elevationFlags`, `slopeArrows`) — included in future project save/load
- Annotation SVG sits above contour SVG (separate element, full opacity)
- Document-level drag handlers for reliable cross-boundary drag
- All helpers (`getSvgPoint`, `computeElevationAt`, `computeSlopeAt`) are stable `useCallback` with empty deps, reading latest state via refs
