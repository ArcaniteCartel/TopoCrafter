## Rendering: Four Z-Order Bands

Curved labels are filtered by exclusive ranges and rendered in the appropriate SVG layer. Each band uses the same pattern: transparent wide hit path ‚Üí textPath text ‚Üí selection anchor circles. The `side='right'` reversal is applied by reversing the points array before computing the Catmull-Rom path.

| zOrder | Layer | SVG element | Location in DOM |
|--------|-------|-------------|-----------------|
| 0‚Äì24 | Below contours | contour SVG | Before sea-level symbol (prior session) |
| 25‚Äì49 | Below roads | annotation SVG | First in annotation SVG, before road `<defs>` |
| 50‚Äì74 | Above POIs (default) | annotation SVG | After POIs block, before measure anchor |
| 75‚Äì100 | Above grid | dedicated SVG | After GridCanvas, inside inner map div |

### Above-Grid SVG Design

- `pointerEvents: mapTool === 'curved-label' ? 'none' : 'auto'` ‚Äî passes through clicks to annotation SVG when drawing tool is active; captures clicks for selection otherwise
- `onClick` on SVG background deselects all
- `onClick` with `stopPropagation()` on hit paths selects the clicked label
- Anchor drag circles use `style={{ pointerEvents: 'auto' }}` with `onMouseDown` setting `labelAnchorDragRef`; coordinate transform still uses `annotation-svg` (same viewBox/dimensions)
